# Sparkle Operations

A complete management tool for cleaning companies — daily deployment, reporting, clients, contracts, earnings, and pending payments.

## Deploy to Vercel

1. Go to https://vercel.com/new
2. Click "Import Third-Party Git Repository" or drag this folder
3. OR upload this whole folder as a new project
4. Click "Deploy"

That's it. No configuration needed.

## Local development

```bash
npm install
npm run dev
```
